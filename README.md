# fcstore
fcstore是一款基于MVP的Android项目，项目中使用了dagger2+rxjava2+okhttp3+retrofit2的架构
#### 采用的开源库包括
* Rxjava2
* RxPermission
* RxBinding
* Dagger2
* Okhttp3
* Retrofit2
* Butterknife
* Glide
